Test MUDs

(Coming soon...)

Phantasmal, as a mudlib, should be generic and extensible enough to support different genres of MUD. So, courtesy of necronomicorp.com, four test MUDs will be hosted, and this should help debug things.

Please visit the MUD index here for more information on the test servers and their operational policies.

Currently, only one test mud has been brought up so far, and already issues are being revealed. Some bugs in phantasmal have already been fixed, and when phantasmal is stable enough, running two different muds off of one phantasmal should help keep out any game-specific creepages from sneaking into phantasmal.

Regarding status, you may assume the following:

  • If the test site is down, the muds are down.
  • If the test site says the mud is down, then it is so.

Design documents for each mud are below:

  • Lovecraft, an oddball default the new admin stumbled across
  • Seas of Night, a ghostly adventure on the high seas.
  • Innsmouth High, a pretend high school set in the Lovecraft genre.
  • Light v. Dark, a power balance game.
