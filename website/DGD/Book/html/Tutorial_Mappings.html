2.8 Mappings

Mappings are data structures, rather like arrays. An array is a sequence of values (often numbers). A mapping is a sort of dictionary of values, so that you can find a value by looking up its key. A mapping is written almost, but not quite, like an array.

The mapping ([ "a" : 1, "b" : 2, "c" : 3 ]) can be dereferenced like an array. But instead of dereferencing by number, this mapping is dereferenced by name. So if a mapping called letters had that value, then letters["a"] would have the value 1 and letters["c"] would have the value 3.
