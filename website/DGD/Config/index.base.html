If You're New To DGD

If you're just getting into DGD, you'll want to know some things. First, you'll need a MUDLib to go with your DGD server. If you don't know what that is, you should definitely read the LPMUD Frequently Asked Questions . And if you don't know what a MUDLib is, you should probably make sure you have plenty of other MUD or programming experience. DGD is an incredible driver with a lot of good features, but it currently has no good MUDLib for a complete beginner to use. It takes a lot of programming to make any good MUD, and DGD starts you off with lots of potential but not much actual game in place.

Don't bother with the add-ons unless you know you specifically need one or another. Don't bother with them at all if you're just getting started. What you mostly want is the DGD server itself. Consult your MUDLib documentation to determine what version of DGD you're likely to need. If you're writing a new MUDLib from scratch, get the most recent experimental version of DGD. They're all pretty stable and Dworkin adds new features constantly.
