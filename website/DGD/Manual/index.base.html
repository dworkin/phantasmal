DGD Reference Manual

The reference manual for DGD comes in 3 volumes.

The first volume concerns the LPC dialect supported by DGD, including a history of the language, details on the workings of the preprocessor, variables, data types, control structures, and functions.

The second volume documents DGD itself, including development history, configuring and building the executable, run time configuration, as well as information on how DGD works.

The third volume documents the Kernel Library, a small public domain mudlib designed as a foundation for development. Motivations for the existence of the kernel library as well as how to use it is included.
