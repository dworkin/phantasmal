Full Subsystems

The components in this section are significant pieces of functionality which may be added to your MUD. They are designed to handle a particular task within the MUD, and handle it entirely, or nearly so.

  • Dworkin, Par Winzell and Erwin Harte put together an Alpha implementation of SSH support for DGD. It works with Melville and the Kernel Library. Phantasmal should include SSHD support based on this package as of Phantasmal-0.017.
  • Erwin Harte's Example Error Manager: errord.c
  • Geir Harald Hansen's Example Object Manager: objectd.c , wiztool_addon.c
  • Jason Cone's Template for InterMUD3 for DGD: Mailing list message
  • John "West" McKenna's (buggy) sample object manager: Mailing list message
  • James Tait's InterMUD3 daemon code, imud3d.c.
  • Another ObjectD, this one by pete at ana dot sk. Here it is.
