Modifying the Default Setup

Phantantasmal comes with a /usr/game directory and a bunch of stuff in it. You can strip that down to the absolute bare bones, or you can build on it. For the "strip to the bare bones" approach, see the "Interface Files" entry from the index page. If you build on it, this document will help you.

There are files in /usr/game/text that are pretty self-explanatory. All of them are simple text files, and you can modify any of them at will. Have fun. You'll need to restart the MUD to see the changed values take effect.
