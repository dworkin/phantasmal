Using Phantasmal

So you've downloaded Phantasmal and gotten it up and running. You're reasonably well-versed in DGD and the Kernel MUDLib (right?). Now it's time to do a little more. You want to know if Phantasmal is for you. Should you use it by itself? Should you build on it?

Yes, of course you should (<grin>). And you should submit your changes to the author or put them up for download as yet another public domain library. But first, you should build some stuff with Phantasmal to get the hang of how to do it.
