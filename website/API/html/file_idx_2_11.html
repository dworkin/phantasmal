API Function: get_level_by_name

Summary:
get the severity level of a string for use with write_syslog.

Defined In LPC Object:
/usr/System/sys/logd.c

Prototype:
mixed get_level_by_name(string severity)

Can Be Called By:
Anybody

Description:

Return a numerical severity for a string. The string is usually one of "warning", "error", "verbose", "debug" or "fatal", though other values may be allowable.

Return Value:

An integer severity, or nil. Nil is returned if the string is not recognized as a severity.

Errors:
Usually none.

See Also:
write_syslog
