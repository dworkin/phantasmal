API Function: language_by_name

Summary:
takes a language name and returns a Phantasmal locale number

Defined In LPC Object:
/usr/System/sys/phrased.c

Prototype:
int language_by_name(string name)

Can Be Called By:
Anybody

Description:

Takes a human-readable language name and returns a Phantasmal locale number. The string can be the language name like "English" or "Spanish", or a locale like "enUS" or "deDE". It could also be one of a couple of simple hybrids like "englishUS" or "espanolES".

Return Value:

An integer indicating the locale, or -1 for no known locale.

Errors:
Usually none.

See Also:
name_for_language , locale_name_for_language
