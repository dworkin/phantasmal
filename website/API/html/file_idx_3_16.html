API Function: num_locales

Summary:
return the number of current locales

Defined In LPC Object:
/usr/System/sys/phrased.c

Prototype:
int num_locales(void)

Can Be Called By:
Anybody

Description:

Returns an integer indicating how many locales are supported by current Phantasmal phrase structures. In theory this could change at runtime, though that rarely happens.

Return Value:

An integer indicating how many locales.

Errors:
Usually none.
