Phantasmal Test Site

Well, here they will be soon. Three muds, each one based on one of Noah's design documents, each to be turned into a fully fledged Phantasmal based MUD.

These MUDs will be maintained for testing, to make sure that Phantasmal remains flexible and usable for widely varying games, and above all else, remains working :). Hopefully, if anything goes wrong with Phantasmal, these test MUDs will be the first to break as an early warning.
