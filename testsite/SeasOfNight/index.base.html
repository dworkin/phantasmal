Bug reporting, downtime

Since this is a test server, it may go down without warning. Please contact shentino@gmail.com if you find that the mud is down for more than 48 hours, and there is no notice on this page about the outage.

However, if there is information about why it's down, or it's marked as being down, please do not bother me. If it goes down with proper notice here, then it was taken down on purpose and will be brought back up ASAP as time permits.
